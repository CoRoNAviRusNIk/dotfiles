## Description

Wayland environment configurations based on Hyprland with a focus on minimalism and custom themes.

---

## Components

### Window Manager

- **Hyprland** - Wayland compositor

### Interface

- **Waybar** - Taskbar
- **Rofi** - Application launcher
- **SwayNC** - Notification center

### Terminal

- **Kitty** - Terminal

### Utilities

- **Fastfetch** - System information
- **Matugen** - Color scheme generator
- **Zathura** - PDF reader

### Themes

- **Custom themes** - Own color schemes and themes located in the `Themes/` directory

- `rofi-tabs.sh` ‚Äî Important categories menu
- `themes_menu.sh` ‚Äî Theme selection menu
- `trigger_menu.sh` ‚Äî Triggers menu
